Original website

Visit laornamental.com

Random local shop that provides driveway gates solutions. Chosen for the horrific state of their webpage.

Usability problems

  • Text color doesn't mix well with the background and makes text hard to read
  • Poorly Structured Links cause visitors to become frustrated
  • Excessive Website Text: content is not concise
  • Lack of Consistency: colors, fonts, graphics and information are not consistent throughout the website
  • Complicated Navigation; navigation content and the interactions are not clear
  • Information Overload; page is full of extraneous elements
  • No mobile support

Accessibility tests and findings

Problems:

  • 116 Contrast Errors
  • 38 X Linked image missing alternative text
  • 33 X Spacer image missing alternative text
  • 2 X Empty link
  • 58 X Redundant link

Overall, hypothetical user with disabilities would have extremely hard time using the website. As lots of features are missing.
